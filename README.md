# Getting Started

To run this project on your local machine, please follow the steps below.

### 1. 📥 저장소 클론

```bash
git clone https://github.com/seulzutora/my-nextjs-app.git
cd my-nextjs-app
```

### 2. 📦 종속성 설치
npm install

### 5. ▶️ 프로젝트 실행
npm run dev

위 명령어를 실행한 후, 웹 브라우저를 열고 다음 주소로 접속하면 프로젝트를 볼 수 있습니다.

http://localhost:3000
